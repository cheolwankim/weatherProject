## 💡 Introducing Project
React를 이용해 오늘의 날씨를 알려주는 웹페이지를 만든 프로젝트입니다.<br>
기상청의 공공 API를 사용하여 서울, 부산, 대구 등 다양한 도시들의 날씨를 한번에 알 수 있습니다<br>
또한 다음날의 날씨를 간략하게 알 수 있습니다.<br>


## ⭐️ How To Use

start:

```sh
npm install
npm start
```
